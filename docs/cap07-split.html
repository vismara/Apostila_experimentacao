9  📖 Capítulo 7 – Delineamento em Parcelas Subdivididas (Split-Plot)

Módulo 2 – Semana 9

9.1 🌱 Narrativa aplicada

Em muitos experimentos florestais, alguns fatores são difíceis de controlar (ex.: espécie, espaçamento de plantio, preparo de área), enquanto outros são fáceis (ex.: adubação na bancada, fungicida por pulverização).
Quando isso acontece, o delineamento em parcelas subdivididas (split-plot) é o caminho natural — ele respeita as restrições operacionais e usa duas fontes de erro (parcela e subparcela) para testar os efeitos apropriadamente.

9.2 🎯 Objetivos

  • Entender por que/quando usar split-plot.
  • Distinguir fator de parcela vs fator de subparcela.
  • Ajustar e interpretar ANOVA com dois erros.
  • Realizar comparações de médias (Tukey) com o ExpDes.pt, preservando os estratos corretos.
  • Traduzir resultado em decisão técnica.

9.3 🧭 Quando é necessário usar parcelas subdivididas?

  • Operação “de difícil ajuste” em larga escala: fator exige mudança de bancada/talhão, preparo de área, ou maquinário (ex.: Espécie, Espaçamento).
  • Custo/tempo elevado ao alternar níveis com frequência: manter um nível por parcela principal e variar o outro dentro dela reduz custo e viabiliza o estudo.
  • Risco/biosegurança: tratamentos que precisam ser mantidos separados (ex.: inoculação, sombreamento fixo).
  • Interferência entre unidades se alternar muito: operações que “vazam” entre unidades vizinhas (ex.: irrigação por setor, controle de pragas por baia).
  • Logística do viveiro/campo: o planejamento obriga a fixar um fator por parcela (whole‑plot) e subdividir para o outro (subplot).
Dica

💡 Regra prática: trate como fator de parcela (whole‑plot) aquilo que não dá (ou não convém) alternar célula a célula; trate como subparcela o que é fácil e frequente de variar.

9.4 🧮 Modelo linear (split-plot)

\[ Y_{ijkl} = \\mu + B_i + A_j + (BA)_{ij} + S_{k(ij)} + C_l + (AC)_{jl} + (BC)_{il} + (ABC)_{ijl} + \varepsilon_{ijkl} \]

  • \(A_j\): fator de parcela (difícil, aplicado nas parcelas principais dentro de cada bloco).
  • \(C_l\): fator de subparcela (fácil, aplicado dentro de cada parcela).
  • \((S_{k(ij)}\): variação aleatória de parcela (erro de parcela).
  • \(\varepsilon\): erro de subparcela.

Regra de ouro: o efeito de A é testado contra o erro de parcela; efeitos de C e interações com C, contra o erro de subparcela.

9.5 🌳 EXEMPLO — Viveiro (Espécie × Adubação) — split‑plot em DIC com ExpDes.pt

Contexto
No viveiro, trocar adubação é simples; trocar de Espécie demanda bancada diferente. Logo, Espécie = parcela, Adubação = subparcela.

Elemento Especificação
Blocos (bancadas) 4 (B1–B4)
Fator de parcela (A) Espécie: Cedrela vs Ipê
Fator de subparcela (C) Adubação: Baixa, Média, Alta
Unidade experimental Bandeja (média de 5 mudas)
Resposta Altura (cm) aos 120 dias

📦 Dados: dados/parcelas_subdivididas_especie_adubacao.csv

9.5.1 💻 ANOVA + Tukey (ExpDes.pt)

Código 
library(readr); library(dplyr); library(ggplot2)
library(ExpDes.pt)

# Quarto local:
dados <- read_csv("dados/parcelas_subdivididas_especie_adubacao.csv", show_col_types = FALSE)
# Colab (se preferir copiar antes p/ /content):
# dados <- read_csv("/content/parcelas_subdivididas_especie_adubacao.csv", show_col_types = FALSE)

dados <- dados |>
  mutate(
    Bloco    = factor(Bloco),
    Especie  = factor(Especie,  levels = c("Cedrela","Ipe")),
    Adubacao = factor(Adubacao, levels = c("Baixa","Media","Alta"))
  )

# Parcela = Especie ; Subparcela = Adubacao
out_viveiro <- psub2.dic(
  fator1     = dados$Especie,       # PARCELA
  fator2     = dados$Adubacao,      # SUBPARCELA
  repet      = dados$Bloco,
  resp       = dados$Altura_cm,
  quali      = c(TRUE, TRUE),
  mcomp      = "tukey",             # letras de Tukey nos estratos corretos
  fac.names  = c("Espécie", "Adubação"),
  sigF       = 0.05,
  sigT       = 0.05,
  unfold     = NULL
)

  geom_boxplot(position = position_dodge(width = 0.8)) +
  labs(title = "Altura – Espécie × Adubação (split-plot, DIC)",
       x = "Adubação", y = "Altura (cm)") +
  theme_minimal()

9.5.2 🧩 Interpretação

  • Espécie é testada com erro de parcela → menor precisão.
  • Adubação e Especie×Adubacao usam erro de subparcela → maior precisão.
  • Se houver interação, a adubação ótima depende da espécie.

9.6 🌾 EXERCÍCIO — Espaçamento × Fungicida em campo jovem

Contexto
Alterar Espaçamento exige maquinário e planejamento (difícil) → parcela. Pulverizar Fungicida é fácil e pode ser feito dentro das parcelas → subparcela. Objetivo: reduzir incidência de doença.

Elemento Especificação
Blocos (talhões) 5 (T1–T5)
Parcela (A) Espaçamento: 2×3 m vs 3×3 m
Subparcela (C) Fungicida: Sem vs Com
Unidade experimental Linha × 10 plantas
Resposta Incidência de doença (%) aos 6 meses

📦 Dados: dados/parcelas_subdivididas_espacamento_fungicida.csv

9.6.1 📊 Orientações d

  • Compare efeitos simples se a interação for significativa (fungicida reduz mais no adensado?).
  • Se a interação não for significativa, reporte efeitos principais (modelo aditivo).
  • Considere a magnitude da redução (%) para decisão de manejo (custo do produto × benefício).

9.7 🧾 “Cola” para o relatório

  1. Justifique o uso do split-plot pela logística do experimento.
  2. Apresente a ANOVA com dois erros (parcela vs subparcela).
  3. Interprete interações com gráficos de médias.
  4. Faça recomendação prática (qual combinação adotar).

9.8 ✅ Síntese

  • Split-plot alinha a estatística às restrições reais.
  • Nem todos os efeitos têm a mesma precisão.
  • Testes certos → recomendações confiáveis.
