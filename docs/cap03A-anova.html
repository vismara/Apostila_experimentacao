4  📖 Capítulo 3A – ANOVA: Introdução e Contexto

Módulo 1 – Semana 3

4.1 🌱 Narrativa inicial

Você é o(a) engenheiro(a) florestal de um viveiro de Eucalyptus grandis. Sua decisão afeta custos e qualidade das mudas que irão para campo.
Para recomendar o tipo de adubação, foi conduzido um teste com três tratamentos:

  • Controle (sem adubo) – referência de custo zero.
  • Orgânica (composto local) – menor custo, possível ganho gradual.
  • Química (NPK 06-30-06) – maior custo, promessa de ganho rápido.

Foram avaliadas 10 mudas por tratamento e medidas Altura (cm), Diâmetro (mm) e Biomassa (g).
👉 Decisão profissional: qual adubação recomendar com base em evidências?

Importante

⚠️ Erro comum: escolher pelo “olhômetro”. Diferenças visuais podem ocorrer ao acaso. A ANOVA formaliza essa comparação de médias.

4.2 🎯 Objetivos de aprendizagem

  • Entender a lógica da ANOVA (variação entre vs. dentro dos grupos).
  • Ler e interpretar a tabela de ANOVA.
  • Aplicar ANOVA no R com dados florestais contextualizados.
  • Conectar gráficos e resumos (Cap. 2) com o teste formal.

4.3 🧠 Intuição da ANOVA

A ANOVA verifica se as diferenças entre as médias dos grupos são maiores do que esperaríamos por acaso.
- Entre grupos (tratamentos): efeito potencial da adubação.
- Dentro dos grupos (erro): variabilidade natural entre mudas do mesmo tratamento.
- Estatística F = QM_trat / QM_erro → quanto maior, maior a evidência de diferença.

4.4 🧾 Tabela de ANOVA (e como ler)

Fonte GL SQ QM = SQ/GL F = QMtrat/QMerro p-valor
Tratamentos k − 1 SQtrat QMtrat F calculado Prob(H0)
Resíduo (Erro) N − k SQerro QMerro — —
Total N − 1 SQtotal — — —
Nota

Como interpretar rapidamente
- GL: graus de liberdade (quantas comparações independentes cabem).
- SQ / QM: variação total e variação média.
- F grande + p-valor pequeno → evidência de diferenças entre médias.

4.5 💻 Aplicação prática (Altura)

Dados novos deste capítulo: dados/eucalipto_adubacao.csv
Conexão com o Cap. 2 (revisão gráfica): dados/crescimento_mudas.csv

Código 
library(readr); library(ggplot2)

euca <- read_csv("dados/eucalipto_adubacao.csv", show_col_types = FALSE)

mod_alt <- aov(Altura_cm ~ Tratamento, data = euca)
summary(mod_alt)
            Df Sum Sq Mean Sq F value  Pr(>F)    
Tratamento   2  154.0   77.02   15.25 3.7e-05 ***
Residuals   27  136.4    5.05                    
---
Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1
Código 
# Visualização
ggplot(euca, aes(Tratamento, Altura_cm, fill = Tratamento)) +
  geom_boxplot() +
  labs(title = "Eucalyptus grandis — Altura por adubação",
       x = "Tratamento", y = "Altura (cm)") +
  theme_minimal()
Dica

✅ Dica prática: Se o p-valor < 0,05, rejeitamos H0 → há evidência de diferença entre os tratamentos. Mas ANOVA não diz quais tratamentos diferem. (isso veremos no Cap. 3B)

Dica

✅ Dica profissional: Relacione o resultado ao custo e logística. Às vezes o ganho de altura é real, mas não compensa o custo do insumo no viveiro.
